# Advanced To-Do Application (TaskFlow)

This project is an advanced task management web application developed as part of Internship Task-04. It allows users to manage tasks efficiently with features like authentication, task prioritization, filtering, and tagging.

---

## Features

* User authentication (Login and Register system)
* Secure user-specific task management using local storage
* Create, update, and delete tasks
* Mark tasks as completed or active
* Set task priority (High, Medium, Low)
* Add deadlines to tasks
* Tag-based task organization
* Advanced filtering:

  * Filter by status (All, Active, Completed, Overdue)
  * Filter by priority
  * Filter by tags
* Search functionality for tasks
* Sorting options:

  * By creation date
  * By deadline
  * By priority
  * By title
* Task statistics dashboard (Total, Active, Completed, Overdue)
* Responsive and user-friendly interface

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage (for data persistence)
